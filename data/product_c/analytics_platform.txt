TROUBLESHOOTING:
Q: My database connection fails?
A: 1) Verify credentials and check the database is online. 2) Whitelist DataFlow IPs: 34.102.140.0/24. 3) Ensure SSL cert is valid. 4) Test connection from Admin > Data Sources > Test.

Q: Dashboard loads slowly?
A: 1) Enable query cache: Dashboard > Settings > Cache > 1 hour. 2) Switch to a data extract (snapshot) for large datasets. 3) Add database indexes on filter columns. 4) Reduce chart refresh interval.

Q: Scheduled reports are not being delivered?
A: Admin > Notifications > SMTP settings. Verify sender domain SPF/DKIM records. Check spam folder. Test: Admin > Reports > Send Test.

Q: How do I create threshold alerts?
A: Dashboard > widget menu > Add Alert > metric, operator, threshold > notification channel (email/Slack/PagerDuty).
